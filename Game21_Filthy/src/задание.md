
## Игра "Очко"

1. Познакомься с кодом игры. Разберись в нём. Отметь места кода, которые тебя особенно раздражают. Места, с которыми было тяжело разобраться.
2. Ознакомься с отзывом на код в файле "code review.md".
3. Ознвкомься с рекомендациями по рефакторингу этого кода.
4. Выполни рефакторинг. (если потребуется помощь, обратись к файлу "clue.md")
5. Сравни результат с кодом в файле "clue.md"
6. Опубликуй проект на GitHub.
